# Phoenix Crime Spatial Analytics
**Geospatial & Exploratory Data Analysis (EDA)**

## The Mission
To uncover actionable insights from urban crime data. This project focuses on deep exploratory data analysis and spatial mapping to identify high-risk zones, temporal patterns, and demographic correlations in the city of Phoenix.

## Technical Architecture
* **Data Wrangling:** Cleaned and standardized a messy, real-world municipal dataset, handling missing values and inconsistent categorical formats using **Pandas**.
* **Temporal Analysis:** Extracted and engineered time-series features to track crime volume fluctuations across different days of the week, months, and seasons.
* **Geospatial Mapping:** Utilized spatial visualization libraries to map coordinate data, identifying distinct crime hotspots and cluster zones.
* **Statistical Visualization:** Designed a comprehensive dashboard of **Seaborn** and **Matplotlib** visualizations to communicate complex trends to non-technical stakeholders.

## Key Insights
* Identified statistically significant geographical clusters where specific crime categories (e.g., property theft vs. violent crime) are concentrated.
* Mapped peak temporal windows for incident reports, providing data-driven recommendations for resource allocation and preventative measures.
